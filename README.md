# 🏟️ StadiumGPT

An AI-powered Smart Stadium Management System designed to improve crowd safety, security monitoring, emergency response, and fan experience during live sporting events.

## 🚀 Overview

StadiumGPT provides stadium operators and visitors with real-time insights about crowd density, occupancy, traffic flow, security status, emergency response readiness, and stadium services.

The platform combines live dashboards, crowd analytics, emergency management tools, and an intelligent fan assistant into a single interface.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/nitink7987/stadium-gpt.git
```

Navigate to the project:

```bash
cd stadium-gpt
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
